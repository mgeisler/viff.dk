equal(self, share_x, share_y)

source code 

Equality testing with secret shared result.

Assumes p = 3 mod 4, returns a secret sharing of 1 if share_x == share_y and 0 otherwise.

This is the probabilistic method based on quadratic reciprocity described in: "Constant-Round Multiparty Computation for Interval Test, Equality Test, and Comparison" by Takashi Nishide and Kazuo Ohta, and fails with probability 1/(2**k) where k is set to the security parameter of the runtime.
