generate_triples(self, field)

source code 

Generate a multiplication triple using PRSS.

These are random numbers a, b, and c such that c = ab. This function can be used in pre-processing.

Returns a tuple with the number of triples generated (1) and a Deferred which will yield a singleton-list with a 3-tuple.
