__nonzero__(self)
(Boolean test operator)

 

Truth value testing.

Returns False if this element is zero, True otherwise. This allows GF elements to be used directly in Boolean formula:

>>> bool(GF256(0))
False
>>> bool(GF256(1))
True
>>> x = GF256(1)
>>> not x
False
